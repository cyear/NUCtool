# 
> [!WARNING]
> 平台支持
> 1. Windows 10
> 2. Windows 11
> 
> 使用教程(支持功能以实际为准)
> 1. 非WIN11系统需检查是否安装webview2
> 2. 首次使用需`调整风扇曲线`并点击`保存配置`
> 3. 每次重新打开程序需要点击`加载配置`
> 4. 点击`Start`
> 5. 配置路径: `%AppData%\com.nuc.x15.fan.cyear.app`
